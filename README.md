# a4c - Automation For C++

## Description

It is noticable that "no code workflow software" are getting all the more popular.
However, such softwares such as _n8n_ have been shown to struggle with large workflows,
lacking in speed and integration with custom code.

a4c (Automation for C++) is one of, if not the first **code-focused** workflow softwares that are **built** in C++.
